# **App Name**: SQL Sentinel

## Core Features:

- Sidebar Navigation: Implement a dynamic, hierarchical sidebar for navigating database features as depicted in the image.
- Database Connection & Selection: Allow users to configure connections to SQL Server instances and seamlessly switch between active databases.
- Table Overview & Alerts: Display a summary of tables for the selected database, highlighting those requiring attention or showing pending issues with a count.
- Performance Monitoring Dashboard: Present key performance indicators (KPIs) and metrics for the active database to visualize its health and operational status.
- AI-Powered Redundancy Scanner: An AI tool that scans database schema and data for redundant structures or information, providing actionable recommendations for optimization and improved database health.
- Maintenance Scheduler: A tool for scheduling archive, backup, and other critical database maintenance operations, with AI providing recommendations for optimal timing based on usage patterns.
- Report Generation & Alert Configuration: Functionality to generate exportable health and performance reports and configure custom alert rules based on predefined or dynamic database metrics.

## Style Guidelines:

- Primary color: Calm, professional blue (#3462AD) used for headings, interactive elements, and key informational displays, symbolizing reliability and data integrity.
- Background color: A very light, subtly cool gray (#F5F6F8) providing a clean, spacious canvas that reduces visual fatigue and keeps focus on content.
- Accent color: A thoughtful, professional purple (#4F378B) to highlight secondary actions, status indicators, and subtle decorative elements, offering clear visual contrast.
- Font: 'Inter' (sans-serif) for both headlines and body text, ensuring modern legibility, a neutral aesthetic, and efficient data presentation suited for technical content.
- Icons: A consistent set of crisp, modern line icons (as observed in the provided image) to offer clear visual cues and enhance navigability within the application.
- Structure: A fixed left-sidebar navigation, meticulously replicating the hierarchy shown in the example, provides consistent access to all database management tools. The main content area dynamically adjusts based on selection.
- Interactions: Subtle and smooth transitions and hover effects for navigation items and clickable components, providing responsive feedback to user actions without introducing distractions.
